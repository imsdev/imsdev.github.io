IBM IMS on GitHub

Welcome to IMS on GitHub. Here you will find sample code and tutorials for your IMS application development and deployment needs.

Application samples

ims-java-jmp

A sample Java app that runs in an IMS dependent region

ims-java-cobol

A sample Java app that inter-operates with COBOL in an IMS dependent region
(Coming soon)

ims-java-jee-tm

Java EE apps that access IMS transactions or process IMS callout requests

ims-java-jee-db

A Java EE app that accesses an IMS database using JDBC and Open database

ims-zcee-api-provider

A tutorial on creating a REST API that accesses an IMS transaction

ims-zcee-api-requester

A tutorial on enabling an IMS application to call out to a REST API

ims-nodejs-tm

A Node.js app that accesses an IMS transaction and runs on Z (Coming soon)

ims-java-springboot

Java samples for accessing IMS transactions or databases from an un-managed, non-Java EE environment, such as Spring Boot with Tomcat.
